Why this needed to be built

The Problem

Managing 100-unit trail events is chaotic

  • —Checkpoints tracked on paper or scattered spreadsheets
  • —No real-time visibility for organizers or participants
  • —Leaderboards updated manually — too slow, error-prone
  • —Safety gaps when field positions are unknown
The Solution

One screen that runs the whole event

  • ✓Instant checkpoint logging from any device
  • ✓Live leaderboard — auto-updated as runners pass through
  • ✓Full organizer dashboard with field visibility
  • ✓Participant profiles with real-time progress tracking
